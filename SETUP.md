# Настройка Isomer Oil Website

## 1. Создание суперпользователя

Выполните команду для создания администратора:

```bash
python manage.py createsuperuser
```

Следуйте инструкциям и создайте логин, email и пароль.

## 2. Настройка Telegram бота

1. Создайте бота через @BotFather в Telegram
2. Получите токен бота
3. Добавьте бота в нужную группу
4. Получите ID группы (можно использовать @userinfobot)
5. Добавьте в файл `.env`:

```
TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_CHAT_ID=id_группы
```

## 3. Начальное наполнение контентом

После входа в админ-панель (http://localhost:8000/admin/):

### 3.1 Информация о компании
- Создайте запись в "Информация о компании"
- Добавьте заголовок, описание и изображение

### 3.2 Возможности компании
- Создайте несколько записей в "Возможности компании"
- Для каждой возможности укажите:
  - Заголовок
  - Описание
  - Иконку (CSS класс, например: "fas fa-oil-can")
  - Порядок отображения

### 3.3 Контактная информация
- Создайте запись в "Контактная информация"
- Добавьте адрес, телефон, email, часы работы

### 3.4 Категории продуктов
- Создайте категории в "Категории"
- Для каждой категории укажите название, описание и изображение

### 3.5 Продукты
- Создайте продукты в "Продукты"
- Укажите название, категорию, описание, характеристики, цену

### 3.6 Партнеры
- Создайте партнеров в "Партнеры"
- Добавьте логотипы, описания, контактную информацию

### 3.7 Записи блога
- Создайте несколько записей в "Записи блога"
- Не забудьте отметить "Опубликовано" для активных записей

## 4. Проверка функциональности

1. Откройте главную страницу: http://localhost:8000/
2. Проверьте все разделы сайта
3. Протестируйте форму обратной связи
4. Протестируйте заказ продуктов
5. Убедитесь, что сообщения приходят в Telegram

## 5. Настройка статических файлов

Для продакшена выполните:

```bash
python manage.py collectstatic
```

## 6. Дополнительные настройки

- Измените настройки в `config/settings.py` для продакшена
- Настройте базу данных PostgreSQL для продакшена
- Настройте веб-сервер (nginx + gunicorn)
